# Attention-Based Time Series Forecasting

## Overview
This project builds an LSTM model with an attention mechanism
to forecast multivariate time series data.

## Features
- Synthetic data generation
- Trend and seasonality modeling
- Regime shift simulation
- LSTM + Attention mechanism
- Baseline comparison (Linear Regression)
- Evaluation using MAE and RMSE

## Tools Used
- Python
- PyTorch
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
